## File Descriptions

- `*_syn.v` - Synthesized netlist
- `*.sdc` - Design constraints
- `*.ddc` - Design Compiler database
- `*.sdf` - Standard Delay Format file
- `testfixture.v` / `tb.sv` - Testbenches
